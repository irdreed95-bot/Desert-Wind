# 📱 White Wolf Store (Desert-Wind) | تطبيق متجر الذئب الأبيض

تطبيق متجر إلكتروني متكامل وتطبيق أندرويد مخصص لبيع أجهزة ومكونات الكمبيوتر والتجميعات الاحترافية في العراق.

An advanced E-Commerce platform and Android application tailored for PC hardware, gaming accessories, and custom builds in Iraq.

---

## 🚀 المميزات الرئيسية | Key Features

- **🛍️ تجربة تسوق سلسة:** استعراض المنتجات والمكونات وأسعارها بدقة.
- **⚙️ لوحة تحكم إدارية:** لوحة تحكم متطورة مخفية خاصة بالإدارة لتعديل وإضافة المنتجات.
- **💬 طلب مباشر:** دعم كامل لنظام الطلب وإرسال الفواتير مباشرة عبر الواتساب (WhatsApp Integration).
- **📦 مظهر عصري:** واجهة مستخدم مخصصة للـ Gamers والـ Setup الاحترافي.

---

## 🛠️ التقنيات المستخدمة | Tech Stack

* **Frontend:** React / TypeScript / Vite
* **Styling:** Tailwind CSS
* **Build Tool:** GitHub Actions (Automated Android APK Build)
* **Database/Storage:** IndexedDB / LocalStorage
